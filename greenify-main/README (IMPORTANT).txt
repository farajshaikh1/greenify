To run this project in the browser, follow the steps below:

1. Download and install the latest version of XAMPP using the link - https://www.apachefriends.org/download.html
2. Move the project folder to - {Installation directory of XAMPP} > xamppfiles > htdocs.
3. Launch XAMPP (or 'manager-osx' on macOS), and under 'Manage Servers' start 'MySQL Database' and 'Apache Web Server'.
4. Type 'localhost/' in the address bar of your browser and hit enter.
5. Open the project folder from the page and navigate to 'Greenify Project' to land on the homepage.
